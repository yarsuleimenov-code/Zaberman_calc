Цель страницы

My Drafts используется для хранения и продолжения работы с quote, которые ещё не были отправлены клиенту.

Это снижает риск потери расчётов и позволяет брокеру возвращаться к незавершённым заявкам без повторного заполнения данных.

Основной сценарий

  • Брокер начал quote, но не завершил отправку estimate.
  • Система автоматически сохранила draft.
  • Брокер открывает My Drafts.
  • Выбирает нужный draft.
  • Продолжает редактирование на основном экране калькулятора.

Будущая логика

  • Auto-save каждые 15–30 секунд.
  • Version history для отправленных estimate.
  • Draft ownership по брокеру.
  • Duplicate draft для похожих заказов.
  • Soft delete вместо полного удаления.
