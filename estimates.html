Цель страницы

My Estimates используется для работы с уже сформированными и отправленными клиенту estimate. Это не черновики, а customer-facing версии quote.

Разница между Draft и Estimate

  • Draft: рабочий черновик, который можно свободно редактировать.
  • Estimate: отправленная клиенту версия, которая должна иметь versioning.
  • После отправки estimate изменения должны создаваться как новая версия.

Основные действия

  • View: открыть отправленную версию estimate.
  • Send Again: повторно отправить клиенту.
  • Reopen Draft: создать редактируемую копию для изменений.
  • Convert: превратить approved estimate в invoice.

Будущая логика

  • PDF / link generation.
  • Email / SMS sending.
  • Estimate expiration rules.
  • Client view tracking.
  • Estimate to Invoice через Stripe / Square.
  • Version history для всех изменений после отправки.
